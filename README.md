ArjunSaiCodeIt3.0 – Astronaut Fitness Checker

Astronaut Fitness Checker is a Flask web application with database support that allows users to log exercises, meals, and daily fitness summaries. The app provides a sleek, browser-based interface to manage fitness tracking efficiently.

This project was submitted by Arjun Sai, a grade 8 student from Canada, for eduSeed's CODE IT 3.0 Competition. Arjun has been continuously learning at eduSeed’s [online coding class](https://eduseed.in/coding/). He is an intelligent and curious student, always coming up with insightful questions and demonstrating a quick learning ability.
